# Namma Mistri - Construction Material Calculator

**Namma Mistri** is a professional Android utility designed for contractors, masons, and homeowners to simplify construction planning. It provides accurate material estimations, team management tools, and project photo tracking in one seamless interface.

---

## ✨ Key Features
- **Material Calculator**: Instant estimation of bricks, cement, sand, and concrete for walls, slabs, and columns.
- **Team Management**: Centralized directory to manage worker roles, attendance tracking, and contact information.
- **Project Photos**: Built-in gallery system to document site progress and organize construction photos.
- **Daily Rates**: Tracking system for local market prices of essential construction materials.
- **Modern UI/UX**: Built with **Material Design 3**, featuring a clean, responsive Edge-to-Edge interface.

## 🛠 Tech Stack
- **Language**: Kotlin
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Persistence Library (Offline-first local storage)
- **UI Components**: ViewBinding, RecyclerView, Material 3 Components
- **Concurrency**: Kotlin Coroutines for high-performance background processing
- **Jetpack Libraries**: Lifecycle (ViewModel, LiveData), Fragment-KTX, WindowManager

## 🏃 How to Run the Project
To evaluate this project locally, follow these steps:

1. **Clone the repository**:
2. **Open in Android Studio**:
   - Open **Android Studio (Ladybug 2024.2.1 or newer)**.
   - Select the `NammaMistri` folder.
3. **Configuration**:
   - Ensure you have **JDK 17** configured in Project Structure.
   - Android SDK Build-Tools **34** or **35** is required.
4. **Sync & Run**:
   - Click **Sync Project with Gradle Files**.
   - Run the app on an emulator or a physical device (Android 7.0 / API 24+).
